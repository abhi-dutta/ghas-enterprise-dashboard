# Screenshots

Place your dashboard screenshots here. Recommended filenames:

- `login.png` — Login page
- `overview.png` — Security overview / landing page
- `overview-heatmap.png` — Organization risk heatmap (close-up)
- `dependabot-charts.png` — Dependabot charts view
- `dependabot-table.png` — Dependabot alerts table view
- `secrets-charts.png` — Secret scanning charts view
- `secrets-table.png` — Secret scanning alerts table view

## How to capture

1. Open the dashboard at `http://localhost:3000`
2. Use your browser's dev tools (Cmd+Shift+P → "Capture full size screenshot") or a screenshot tool
3. Save the images in this folder with the filenames above
